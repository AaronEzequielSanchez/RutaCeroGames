# Rutacero Games

Proyecto web desarrollado como trabajo académico para la facultad.

Rutacero Games es un sitio web de un estudio indie ficticio dedicado al desarrollo
de videojuegos con identidad argentina, narrativa oscura y fuerte enfoque cultural.

---

## 👤 Autor

- **Nombre:** Ezequiel Sanchez  
- **Rol:** Diseño y desarrollo web  
- **Año:** 2026  

---

## 🎮 Descripción del proyecto

El sitio representa a un estudio de videojuegos independiente e incluye:

- Página de inicio con presentación del estudio
- Sección de juegos con sistema de carrito
- Página de contacto con formulario
- Carrito de compras funcional
- Diseño responsive (adaptado a celular)
- Estética oscura inspirada en el folclore argentino

El objetivo del proyecto es aplicar conocimientos de HTML, CSS y JavaScript
en un sitio web funcional y bien estructurado.

---

## 🛠️ Tecnologías utilizadas

- **HTML5** – estructura del sitio  
- **CSS3** – diseño, animaciones y responsive  
- **JavaScript** – interactividad y carrito de compras  
- **LocalStorage** – persistencia del carrito  
- **Google Fonts** – tipografías Cinzel y Montserrat
